
```
git clone https://github.com/KadoBOT/pictabite.git
cd pictabite
npm install
replace files from node_modules/aphrodite/ with these files found here(https://github.com/Khan/aphrodite)
```

Development Mode:
```
npm start
open localhost:3000
```
Production Mode:
```
npm run build
open locally with any http-server
```
